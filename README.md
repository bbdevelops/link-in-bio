# Static Link Landing Page

Uses HTML/CSS for a simple static LinkTree-esque site to use as a simple way to distribute social media links and other interesting items.

Customize the index.html file with your personal social media and items of interest in a code/text editor of your choice. Also you'll need some images of your own, you'll have to be creative. The social media icons use FontAwesome. Use their search site to find free social media icons to use. Run in a browser of your choice.
